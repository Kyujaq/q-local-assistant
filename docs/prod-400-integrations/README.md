# Assistant External Integrations

This folder contains specifications for **external system integrations** used by the q-local-assistant runtime.

## What Goes Here
- Home Assistant integration patterns and APIs
- Paprika recipe/pantry/grocery integration
- Camera and vision service integrations
- Tailscale satellite connectivity
- Email, calendar, and other approved external services
- Authentication and security patterns
- API wrappers and integration layers

## What Doesn't Go Here
Development tool integrations (git, testing frameworks) belong in `dev-150-workflow/`.
